# Roadmap

## Phase 1: Foundation

- Finalize product requirements
- Define service contracts and schema boundaries
- Scaffold monorepo structure
- Set up Docker Compose for local PostgreSQL, Redis, and mail testing

## Phase 2: Identity and core data

- Implement auth service
- Implement member CRUD
- Implement attendance flow
- Implement plan assignment and offer visibility

## Phase 3: Automation

- Add expiry reminder jobs
- Add audit logs
- Add operational dashboards

## Phase 4: Product hardening

- Add tests
- Add observability
- Add rate limiting and security hardening
- Prepare deployment manifests
